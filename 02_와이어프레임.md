## 🎨 컴포넌트별 상세 디자인

### 1. 로또볼 컴포넌트
```css
/* 번호별 색상 규칙 */
1-10:   빨간색 (#FF4444)
11-20:  주황색 (#FF8800) 
21-30:  노란색 (#FFDD00)
31-40:  파란색 (#0088FF)
41-45:  보라색 (#8844FF)
보너스:  금색 (#FFD700)
```

### 2. 버튼 디자인
```
기본 버튼:
┌─────────────────┐
│ 🎯 AI 번호 생성  │  (배경: 그라데이션, 그림자 효과)

┌─────────────────┐
│    회차 검색     │  (테두리 스타일)

```

## 🔄 인터랙션 플로우

### 1. AI 번호 생성 플로우
```
[생성 버튼 클릭] 
    ↓
[로딩 애니메이션 3초]
    ↓
[번호 애니메이션으로 표시]
    ↓
[복사/저장 옵션 제공]
```

### 2. 회차 검색 플로우
```
[회차 입력]
    ↓
[실시간 유효성 검사]
    ↓
[결과 슬라이드 애니메이션]
    ↓
[당첨금액/당첨자수 표시]
```

### 3. 반응형 네비게이션
```
데스크톱: 가로 메뉴바
모바일: 햄버거 메뉴 → 슬라이드 메뉴
```
