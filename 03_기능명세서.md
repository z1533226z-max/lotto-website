# 03. 기능명세서 (2026-03 최신)

## 서비스 개요
**로또킹** (https://lotto.gon.ai.kr) - AI 기반 로또번호 추천 및 분석 서비스

## 핵심 기능

### 1. 로또 당첨번호 조회
- **최신 당첨번호** (`/`): 메인 페이지에 최신 회차 결과 표시
- **회차 상세** (`/lotto/[round]`): 특정 회차 당첨번호, 당첨금, 판매점 정보
- **최근 결과** (`/lotto/recent`): 최근 10회차 결과 목록
- **전체 목록** (`/lotto/list`): 전체 회차 검색/페이지네이션
- **데이터 소스**: gon-services API (primary) → 정적 데이터 → smok95 API → 동행복권 (4단계 fallback)

### 2. AI 추천번호 생성
- **AI 번호 생성기** (`/` 메인): 통계 가중치 기반 번호 생성 (1~5세트)
- **AI 적중 기록** (`/lotto/ai-hits`): 과거 AI 예측 vs 실제 결과 비교
- **Gemini API 연동**: Google AI로 예측 번호 생성
- **사용량 제한**: 비회원 3회/일, 회원 무제한

### 3. 통계 분석
- **종합 통계** (`/lotto/statistics`): 번호 출현빈도, 핫/콜드, 트렌드
- **번호별 분석** (`/lotto/number/[id]`): 1~45번 개별 번호 출현 패턴
- **연도별 분석** (`/lotto/year/[year]`): 2002~현재 연도별 당첨번호 통계
- **패턴 분석** (`/lotto/pattern/[type]`): 홀짝, 고저, 합계, 연번, 구간, 끝수, 갭, AC값

### 4. 특수 번호 생성
- **꿈해몽** (`/lotto/dream`): 꿈 키워드 → 번호 변환 (개별 상세: `/dream/[keyword]`)
- **행운번호** (`/lotto/fortune`): 생년월일 기반 행운 번호

### 5. 도구
- **시뮬레이터** (`/lotto/simulator`): 로또 구매 시뮬레이션 (투자금액 대비 수익)
- **계산기** (`/lotto/calculator`): 당첨금 세금 계산
- **가이드** (`/lotto/guide`): 로또 완전 가이드

### 6. 당첨 판매점
- **판매점 조회** (`/lotto/stores`): 회차별/지역별 당첨 판매점 검색
- **지역 통계**: 시/도별, 구/군별 당첨 집중도 분석
- **데이터**: Supabase winning_stores 테이블

### 7. 커뮤니티
- **게시판** (`/community`): 자유/예측/후기/팁 카테고리
- **게시글 CRUD**: 비밀번호 기반 (비회원도 작성 가능)
- **댓글/대댓글**: 중첩 댓글 지원
- **좋아요**: 게시글/댓글 좋아요
- **관리자 모드** (`/admin`): 게시글 관리, 사용자 차단

### 8. 회원 시스템
- **닉네임+비밀번호** 인증 (소셜 로그인 없음)
- **마이페이지** (`/mypage`): 저장번호, 당첨확인, 통계, 뱃지
- **번호 저장**: AI/꿈해몽/행운번호 저장 (최대 100건, FIFO)
- **당첨 확인**: 저장번호 vs 실제 결과 자동 매칭

### 9. 게이미피케이션
- **뱃지 시스템**: 방문 스트릭, AI 사용, 번호 저장 등 업적 달성
- **리더보드**: 최장 연속 방문, 뱃지 수 순위
- **트렌드 알림**: 최근 트렌드 변화 알림
- **개인화 인사이트**: 사용자 활동 기반 맞춤 분석

### 10. SEO & 수익화
- **구조화 데이터**: JSON-LD (WebApplication, FAQPage)
- **동적 사이트맵**: 전 페이지 URL 자동 생성
- **AdSense**: 반응형 광고 배치

## DB 스키마 (Supabase)

| 테이블 | 설명 |
|---|---|
| `winning_stores` | 당첨 판매점 (회차, 순위, 매장명, 주소, 구매유형) |
| `posts` | 커뮤니티 게시글 |
| `comments` | 댓글 (대댓글 parent_id) |
| `user_profiles` | 회원 프로필 |
| `user_progress` | 게이미피케이션 진행상황 (스트릭, 뱃지, 카운터) |
| `saved_numbers` | 저장 번호 히스토리 (source: ai/dream/fortune) |
| `rate_limits` | IP 기반 요청 제한 |
| **뷰** `posts_with_comment_count` | 게시글 + 댓글 수 |
